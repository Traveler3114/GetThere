# EF Core Migration Guide

This solution has two databases, each owned by its respective API:

| Database | Owner | Connection |
|----------|-------|------------|
| **GetThereDB** | GetThereAPI | `Server=localhost;Database=GetThereDB;Trusted_Connection=True;TrustServerCertificate=True` |
| **TransitInfoDB** | TransitInfoAPI | `Server=localhost;Database=TransitInfoDB;Trusted_Connection=True;TrustServerCertificate=True` |

---

## First Time Setup

Install EF Core tools globally (once):
```bash
dotnet tool install --global dotnet-ef
```

---

## Every Time You Add or Change a Table

### Step 1 — Stop the API
Press `Shift + F5` or click the red Stop button

### Step 2 — Open Terminal
`View → Terminal`

### Step 3 — Navigate to the API project

For GetThereAPI:
```bash
cd GetThereAPI
```

For TransitInfoAPI:
```bash
cd TransitInfoAPI
```

### Step 4 — Create the migration
```bash
dotnet ef migrations add YourMigrationName
```

Use a descriptive name every time, for example:
- `InitialCreate` — first time ever
- `AddedWalletTable` — when you add a Wallet table
- `AddedRouteShape` — when you add a shape column

### Step 5 — Apply migration
```bash
dotnet ef database update
```

> **Note:** TransitInfoAPI auto-runs `MigrateAsync()` on startup, so Step 5 is optional if you restart the API — but running `database update` explicitly is still recommended.

### Step 6 — Start the API again
Press `F5`

## Rules to Remember

- Always **stop the API** before running migrations
- Always run commands from inside the **correct project folder**
- Migration names must be **unique** every time
- Always run **both commands** — `migrations add` creates the file, `database update` applies it to SQL Server
- Never manually edit `*ModelSnapshot.cs` — auto-generated by EF Core

---

## Adding a New Table — Full Process

1. Create a new model class in `Entities/`
2. Add `DbSet<YourModel>` to the `DbContext`
3. Stop the API (`Shift + F5`)
4. `cd` into the project folder
5. Run `dotnet ef migrations add <Name>` + `dotnet ef database update`
6. Start the API (`F5`)
